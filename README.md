# Página de presentación — intro

Esta rama contiene una página estática de presentación con diseño mejorado y tipografía legible.

Archivos añadidos:
- `index.html` — página principal (HTML).
- `styles.css` — estilos CSS (responsive, paleta clara y accesible).
- `README.md` — instrucciones básicas.

Cómo publicar en GitHub Pages:
1. Asegúrate de que los archivos están en la rama `main` en la raíz del repositorio.
2. Ve a la configuración del repositorio en GitHub -> Pages.
3. En "Build and deployment", selecciona la rama `main` y la carpeta `/ (root)`.
4. Guarda. GitHub generará la URL pública (normalmente `https://<usuario>.github.io/<repo>`).

Previsualizar localmente:
- Con Python 3 instalado, desde la raíz del repo:
  - `python -m http.server 8000`
  - Abre `http://localhost:8000` en tu navegador.

Siguientes pasos que puedo hacer si quieres:
- Subir una foto real (avatar.jpg) y favicon.
- Añadir meta tags Open Graph para compartir en redes.
- Añadir formulario de contacto o enlace a LinkedIn.
